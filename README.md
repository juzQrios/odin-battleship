## Get Started

### Setup

1. Clone the repository.

```bash
git clone git@github.com:juzQrios/odin-battleship.git
cd odin-battleship
```

2. Install dependencies.

```bash
npm install
```

3. Build source code.

```bash
npm run build-prod
```

4. Open `index.html` inside `/dist` folder in your browser.

### Tests

```bash
npm run test a
```

### Deployment

Deploy `/dist` folder after building source code.
